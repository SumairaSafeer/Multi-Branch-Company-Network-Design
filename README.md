## Featured Projects
### 1. Company / Business System Network Design (Trading Floor Support Center)

**Description**:  
Complete hierarchical (Core-Distribution-Access) network design for a 600-user trading floor relocating to a new three-floor building. Features full redundancy at every layer, dual-ISP connectivity, per-department VLANs/WLANs, OSPF routing, DHCP, port-security, and PAT.

**Key Technologies**:
- Hierarchical topology with multilayer switches
- OSPF dynamic routing + dual-homed routers
- Per-department VLANs (10, 20, 30, 40, 50, 60)
- Port security on Finance & Accounts
- SSH remote management

### 2. Multi-Branch Company Network Design with Centralized DHCP, HTTP & DNS
**Description**:  
Multi-site enterprise network connecting two branches and a central server site. Demonstrates centralized service delivery over WAN using RIP, mixed addressing strategies (dynamic DHCP + static), wireless access, VLANs, and inter-VLAN routing.

**Key Technologies**:
- Hub-and-spoke topology with RIP v2
- Centralized DHCP/HTTP and DNS services
- VLSM subnetting across branches
- DHCP relay (`ip helper-address`)
- Wireless departments + Finance VLAN (router-on-a-stick)
